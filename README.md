## About XTools

XTools combines a practical set of day-to-day developer helpers in one browser app, with JSON inspection, formatting, validation, diffing, and structured visualization at the center. Because it is meant for quick access in the web, it fits well when you want to review local data, compare payloads, or inspect nested content without installing a separate desktop tool.

For Salesforce work, it also supports Apex and LWC source exploration through class diagram generation and related flow views. On top of that, the Apex debug log analyzer makes it possible to parse and review trace output. With local file and folder support, multiple tabs, and saved history, XTools works well for repeat analysis and side-by-side comparisons.

## Usage

A common XTools session follows a straightforward flow:

1. Load a JSON file, paste content into the editor, or open a folder-based source.
2. Switch to the tool you need, such as formatter, validator, grid viewer, or diff.
3. For Salesforce code, generate a class diagram or flow visualization from Apex or LWC sources.
4. For logs, import an Apex debug trace and inspect the parsed output.
5. Use search, filtering, tabs, and history to move between related items quickly.

When comparing data, load both inputs and open the diff view for a side-by-side check. For larger outputs, the grid and parser views make nested structures easier to explore while keeping context intact.
